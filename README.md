## 项目描述


- 基于React、Redux、Node.js与MongoDB搭建的低代码平台，支持PC端和安卓移动端的多端使用。

- 用户可以通过拖拽组件定义组件的布局，系统会对组件在画布上进行实时渲染。
 
- 导出格式分为代码导出和表单导出，用户可以根据自己的需求来进行导出。
